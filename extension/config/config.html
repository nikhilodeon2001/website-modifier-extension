ℹ️ How to Use

  1. Get your OpenAI API key from platform.openai.com/api-keys
  2. Paste your API key above and click "Save Settings"
  3. Navigate to any webpage (e.g., CNN, ESPN, NYTimes)
  4. Click the extension icon in your browser toolbar
  5. Enter a topic and click "Transform Page"
  6. Watch as the page content is replaced with AI-generated content!

⚠️ Important Notes

  • API Costs: Using this extension will make API calls to OpenAI which may incur costs based on your usage.
  • Rate Limits: Be aware of OpenAI's rate limits. Image generation may take time.
  • Privacy: Your API key is stored locally in your browser and never sent anywhere except OpenAI.
  • Page Refresh: Transformations are temporary. Refresh the page to restore original content, or use the "Reset Page" button.
